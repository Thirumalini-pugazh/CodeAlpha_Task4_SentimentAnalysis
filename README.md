# Sentiment Analysis

## Project Title
Customer Review Sentiment Analysis

## Objective
The objective of this project is to analyze customer reviews and classify them into Positive, Negative, or Neutral sentiments using Natural Language Processing (NLP) techniques.

## Tools and Technologies Used
- Python
- Pandas
- NLTK
- TextBlob
- Matplotlib
- Jupyter Notebook / VS Code

## Dataset Description
The dataset contains customer reviews collected from online platforms. Each review represents customer opinions about products or services.

### Dataset Columns
- Review ID
- Review Text
- Sentiment

## Analysis Performed

### 1. Data Collection
- Loaded customer review data from a CSV file.

### 2. Data Preprocessing
- Removed special characters and unwanted symbols.
- Converted text into lowercase.
- Cleaned and prepared the data for analysis.

### 3. Sentiment Analysis
- Classified reviews as:
  - Positive
  - Negative
  - Neutral

### 4. Data Visualization
- Created charts showing sentiment distribution.
- Compared the number of positive, negative, and neutral reviews.

## Key Findings

- Most reviews were Positive.
- A smaller number of reviews were Negative.
- Neutral reviews indicated mixed customer opinions.
- Overall customer satisfaction was high based on sentiment scores.

## Output

The project generated:
- Sentiment classification results
- Summary statistics
- Sentiment distribution charts
- CSV file with predicted sentiments

## Conclusion

Sentiment Analysis helps organizations understand customer opinions and improve products and services based on feedback. This project demonstrates how NLP techniques can be used to extract valuable insights from textual data.
